Help

Vapid is a work in progress. If you have any questions, or would like to help make it better, please use one of the channels below.

Using the Software

If you are having difficulty using Vapid, please submit questions to the Vapid forums. Additionally, if you are someone that feels confident using Vapid, please consider answering questions there too.

Bugs and Open Source

If you encounter a bug, or would like to participate in Vapid’s development, please join in at https://github.com/vapid/vapid.

Improve This Document

This website can most certainly be improved. If you notice anywhere that could use a spelling, grammar, explanatory, or other improvement, please submit a pull request or open an issue on https://github.com/vapid/vapid-docs.
